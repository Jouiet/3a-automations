## TENDANCES ANIMATION 2025 (VERIFIE WEB)

### 1. Micro-Interactions Intelligentes
> "Micro-interactions are becoming smarter and more personalized. They help users understand system feedback instantly while making the experience feel alive."
- Source: Multiple 2025 trend reports

### 2. Scroll-Triggered Animations (ScrollTrigger)
> "Scroll-triggered motion remains one of the biggest JavaScript animation trends in 2025."
- GSAP ScrollTrigger est gratuit depuis 2025 (acquisition Webflow)
- Scrub, pinning, markers pour debug

### 3. Purposeful Motion Design
> "2025 marks a shift toward more intentional and refined motion design. Designers are embracing a 'less is more' philosophy."
- Moins d'animations excessives
- Chaque animation doit avoir un objectif

### 4. Liquid Motion / Morph Effects
> "Liquid motion design effects use smooth, flowing movements to create visually captivating transitions."
- Morphing shapes
- Gradient color shifts
- Ripple effects

### 5. SVG Animations
> "More designers are leveraging SVG over traditional image formats for lightweight, scalable animations."
- SVG peut etre anime avec HTML/CSS
- File-size-friendly

## PROMPT 5: CLAUDE/GPT - ANIMATIONS CSS/JS

### 5A. Scroll-Triggered Animations System

```
Tu es un expert JavaScript specialise en animations scroll performantes.

MISSION: Cree un systeme complet d'animations declenchees au scroll pour une landing page.

CONTEXTE TECHNIQUE:
- GSAP ScrollTrigger est gratuit depuis 2025 (acquisition Webflow)
- Ou utiliser IntersectionObserver natif pour zero-dependency
- Doit fonctionner sans framework

ELEMENTS A ANIMER:
1. Section headers: fade-up (translateY: 30px → 0)
2. Cards: staggered entrance (0.1s delay entre chaque)
3. Process steps: reveal sequentiel gauche-droite
4. Stats: count-up animation quand visible
5. CTA: scale-in depuis 0.95

CONFIGURATION SCROLL:
- Threshold: 0.2 (20% visible)
- Root margin: "-50px"
- Animation once: true

CLASSES CSS REQUISES:
.reveal - Initial state (opacity: 0, transform)
.reveal.active - Final state (opacity: 1, transform: none)
.reveal-up, .reveal-left, .reveal-right, .reveal-scale

PERFORMANCE:
- will-change sur elements animes
- transform et opacity uniquement (GPU)
- prefers-reduced-motion: disable animations

Genere:
1. CSS complet avec keyframes et classes
2. JavaScript avec IntersectionObserver
3. Exemples d'usage HTML
```

### 5B. 3D Card Tilt Effect

```
Cree un effet de tilt 3D sur des cards au hover, style 2025.

Specifications:
- Rotation max: 15deg sur X et Y
- Perspective: 1000px
- Transition smooth au retour
- Glare effect: overlay blanc qui suit le curseur
- Performance: will-change, transform3d

Couleurs:
- Card background: #1B2F54
- Border glow au hover: #4FBAF1
- Glare: rgba(255,255,255,0.1)

Contraintes:
- Vanilla JavaScript (pas de bibliotheque)
- Touch devices: desactiver tilt, garder glow
- Accessible: pas de mouvement si reduced-motion

Code HTML, CSS et JS complet.
```

### 5C. Gradient Text Shimmer

```
Cree un effet shimmer anime sur du texte avec gradient.

Texte: "solutions sur mesure"
Couleurs: #E4F4FC → #ADD4F0 → #4FBAF1

Effet: Highlight lumineux qui traverse le texte de gauche a droite.

Technique CSS:
- background: linear-gradient
- background-clip: text
- -webkit-text-fill-color: transparent
- animation: shimmer sur background-position
- Duree: 3s, infinite

Code CSS complet avec fallback pour navigateurs anciens.
```

### 5D. Glassmorphism Header

```
Cree un header avec effet glassmorphism moderne 2025.

Specifications:
- Position: fixed
- Background: rgba(25, 30, 53, 0.8)
- Backdrop-filter: blur(20px) saturate(180%)
- Border-bottom: 1px solid rgba(79, 186, 241, 0.1)

Comportements:
1. Scroll > 50px: header devient plus opaque
2. Scroll down: header se cache (translateY: -100%)
3. Scroll up: header reapparait
4. Transition smooth sur tous les changements

Elements:
- Logo: "3A" cyan + "Automation" gris
- Nav links: Services, Process, Contact
- CTA button: "Contact" avec background cyan

Mobile:
- Hamburger icon (3 barres)
- Animation morph en X au clic
- Menu fullscreen overlay

Code HTML, CSS, JS complet.
```
