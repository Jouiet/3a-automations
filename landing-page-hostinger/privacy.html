Politique de confidentialité

Dernière mise à jour : 17 Décembre 2025

1. Introduction

3A Automation ("nous", "notre", "nos") s'engage à protéger la confidentialité de vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous visitez notre site web 3a-automation.com.

2. Données collectées

2.1 Données de navigation

Nous collectons automatiquement certaines informations techniques :

  • Adresse IP (anonymisée)
  • Type de navigateur et version
  • Pages visitées et temps passé
  • Source du trafic (référent)
  • Appareil et système d'exploitation

2.2 Données personnelles

Lorsque vous utilisez notre formulaire de contact, nous collectons :

  • Nom
  • Adresse email
  • URL de votre site web
  • Type de service demandé
  • Message (optionnel)

3. Utilisation des données

Vos données sont utilisées pour :

  • Répondre à vos demandes d'audit ou de contact
  • Améliorer notre site web et nos services
  • Analyser le trafic et les performances du site
  • Personnaliser votre expérience
  • Se conformer aux obligations légales

4. Cookies et technologies de suivi

Nous utilisons les technologies suivantes :

4.1 Cookies essentiels

Nécessaires au fonctionnement du site (préférence de consentement).

4.2 Cookies analytiques (Google Analytics 4)

Si vous acceptez, nous utilisons GA4 pour comprendre comment les visiteurs interagissent avec notre site. Ces données sont anonymisées (IP masquée).

4.3 Google Tag Manager

Utilisé pour gérer les scripts de suivi de manière centralisée. Aucune donnée n'est collectée directement par GTM.

4.4 Vos choix

Vous pouvez accepter ou refuser les cookies non-essentiels via notre bannière de consentement. Votre choix est sauvegardé pendant 365 jours.

5. Partage des données

Nous ne vendons jamais vos données. Elles peuvent être partagées avec :

  • Google Analytics - Analyse du trafic (si vous consentez)
  • Hostinger - Hébergement du site
  • n8n (auto-hébergé) - Traitement des formulaires

6. Sécurité des données

Nous mettons en œuvre des mesures de sécurité appropriées :

  • Connexion HTTPS chiffrée
  • Serveurs sécurisés
  • Accès limité aux données
  • Pas de stockage de données sensibles

7. Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès - Obtenir une copie de vos données
  • Droit de rectification - Corriger vos données inexactes
  • Droit à l'effacement - Supprimer vos données
  • Droit à la portabilité - Recevoir vos données dans un format standard
  • Droit d'opposition - Refuser le traitement de vos données
  • Droit de retrait du consentement - À tout moment

Pour exercer ces droits, contactez-nous à : contact@3a-automation.com

8. Conservation des données

  • Données de formulaire - 3 ans après le dernier contact
  • Données analytiques - 14 mois (paramètre GA4)
  • Cookies de consentement - 365 jours

9. Transferts internationaux

Certaines données peuvent être transférées vers des serveurs situés hors de l'UE (Google, États-Unis). Ces transferts sont encadrés par les Clauses Contractuelles Types de la Commission Européenne.

10. Modifications

Nous pouvons modifier cette politique à tout moment. Les changements seront publiés sur cette page avec la date de mise à jour.

11. Contact

Pour toute question concernant cette politique :

  • Email : contact@3a-automation.com
  • Site web : 3a-automation.com
