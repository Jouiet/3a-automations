Démarrer avec 3A Automation

30 minutes
4 leçons

Les bases pour bien commencer : connectez vos comptes, découvrez le tableau de bord et lancez votre première automation.

1 Accéder à votre espace client 5 min

Après votre souscription à un pack 3A Automation, vous recevez un email avec vos identifiants de connexion.

Étape 1 : Connexion

Rendez-vous sur dashboard.3a-automation.com et entrez votre email et mot de passe temporaire.

Étape 2 : Changez votre mot de passe

Pour des raisons de sécurité, modifiez immédiatement votre mot de passe dans les paramètres du compte.

Conseil

Utilisez un mot de passe fort avec au moins 12 caractères, incluant lettres, chiffres et symboles.

2 Connecter votre compte Klaviyo 8 min

Klaviyo est votre plateforme d'email marketing. La connexion permet de synchroniser vos contacts et d'envoyer des emails automatisés.

Étape 1 : Obtenir votre clé API

Dans Klaviyo, allez dans Account > Settings > API Keys. Créez une nouvelle clé privée avec les permissions "Full Access".

Étape 2 : Ajouter dans 3A Automation

Dans votre tableau de bord 3A, allez dans Intégrations > Klaviyo. Collez votre clé API et cliquez sur "Connecter".

Étape 3 : Vérifier la synchronisation

Un message de confirmation apparaît. Vos listes Klaviyo sont maintenant visibles dans l'interface.

Conseil

Ne partagez jamais votre clé API. Elle donne accès complet à votre compte Klaviyo.

3 Connecter votre boutique Shopify 8 min

La connexion Shopify permet de synchroniser vos produits, commandes et clients pour des automations e-commerce.

Étape 1 : Installer l'app

Dans votre admin Shopify, allez dans Apps. Recherchez "3A Automation" ou utilisez le lien d'installation fourni dans votre email de bienvenue.

Étape 2 : Autoriser l'accès

Shopify affiche les permissions demandées (produits, commandes, clients). Cliquez sur "Installer l'app".

Étape 3 : Synchronisation initiale

La première synchronisation peut prendre quelques minutes selon la taille de votre catalogue.

4 Tour du tableau de bord 9 min

Vue d'ensemble

Le tableau de bord affiche vos métriques clés : emails envoyés, taux d'ouverture, revenus générés.

Sections principales

  • Accueil : Résumé des performances des 7 derniers jours
  • Automations : Liste de vos flux automatisés actifs
  • Contacts : Gestion de vos listes et segments
  • Analytics : Rapports détaillés et exports
  • Intégrations : Connexion de vos outils externes

Prochaines étapes

  • Segmenter vos contacts pour des campagnes ciblées
