Analytics

Piloter par la Data

45 min
5 leçons

Comprendre vos tableaux de bord, identifier les métriques clés et prendre les bonnes décisions basées sur les données.

1 Vue d'ensemble du tableau de bord 8 min

Page d'accueil Analytics

Le tableau de bord affiche un résumé des 7 derniers jours par défaut. Vous pouvez ajuster la période en haut à droite.

Widgets principaux

  • Revenus générés : Chiffre d'affaires attribué aux automations
  • Emails envoyés : Volume total avec taux d'ouverture moyen
  • Leads captés : Nouveaux contacts entrés dans le système
  • Conversion : Taux de passage lead → client

Navigation

Cliquez sur n'importe quel widget pour accéder au rapport détaillé correspondant.

2 Métriques email essentielles 10 min

KPIs à surveiller

  • Taux d'ouverture : % d'emails ouverts. Objectif : > 20%
  • Taux de clic (CTR) : % de clics sur liens. Objectif : > 2%
  • Taux de conversion : % de clics devenus achats
  • Revenu par email (RPE) : CA / nombre d'emails envoyés
  • Taux de désinscription : À garder < 0.5%

Attention aux ouvertures

Depuis iOS 15, les taux d'ouverture sont moins fiables (Apple Mail Privacy). Privilégiez le CTR et le revenu comme indicateurs principaux.

3 Analyser vos automations 10 min

Rapport par automation

Dans Analytics > Automations, comparez la performance de chaque flux :

  • Bienvenue : Objectif 30%+ de conversion en acheteur
  • Panier abandonné : Objectif 10-15% de récupération
  • Post-achat : Objectif 20%+ de rachat

Identifier les problèmes

Si le taux d'ouverture est bon mais le CTR faible, le contenu n'est pas convaincant. Si l'ouverture est faible, l'objet doit être amélioré.

4 Suivi des leads 8 min

Entonnoir de conversion

Le rapport Leads affiche le parcours complet :

  • Sources : D'où viennent vos leads (LinkedIn, Google Maps, formulaires)
  • Qualifiés : % de leads avec score > seuil
  • Contactés : % ayant reçu un email
  • Convertis : % devenus clients

Comparez les sources : laquelle génère le meilleur ratio qualifiés/convertis ?

5 Exporter et partager 9 min

Options d'export

  • PDF : Rapport visuel pour présentation
  • CSV : Données brutes pour analyse avancée
  • Email automatique : Rapport hebdomadaire envoyé chaque lundi

Configurer le rapport automatique

Dans Paramètres > Rapports, activez l'envoi hebdomadaire. Ajoutez les destinataires (vous, votre équipe).

Conseil

Partagez les résultats avec votre équipe. La transparence sur les données motive tout le monde à améliorer les performances.
