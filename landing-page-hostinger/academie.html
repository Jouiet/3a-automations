linkedin-to-klaviyo-pipeline.cjs (Modèle Référence)

Pipeline complet LinkedIn → Klaviyo avec segmentation et emails personnalisés. 920 lignes, testé production.

Fonctionnalités

  • Scraping LinkedIn via Apify (email extraction)
  • Segmentation automatique par 6 personas B2B
  • Push vers Klaviyo avec properties enrichies
  • Event tracking pour trigger de flows
  • Templates email pré-personnalisés stockés
bash
# Dry run avec données test node automations/agency/linkedin-to-klaviyo-pipeline.cjs --test # Scraping LinkedIn live node automations/agency/linkedin-to-klaviyo-pipeline.cjs --search="marketing director morocco" --max=50 # Import depuis fichier JSON node automations/agency/linkedin-to-klaviyo-pipeline.cjs --input=profiles.json --live

Segments B2B Détectés

email-automation-unified.cjs (Welcome + Outreach)

Automation email unifiée : Welcome Series (5 emails) + Outreach Sequence (3 emails par segment). 598 lignes.

bash
# Mode CLI - Welcome Series node automations/agency/email-automation-unified.cjs --mode=welcome --email=client@example.com # Mode CLI - Outreach avec segment auto node automations/agency/email-automation-unified.cjs --mode=outreach --json='{"email":"ceo@company.com","firstName":"Ahmed","company":"TechCorp","title":"CEO"}' # Mode HTTP Server (n8n compatible) node automations/agency/email-automation-unified.cjs --server --port=3001 # Endpoints disponibles: # POST http://localhost:3001/welcome # POST http://localhost:3001/outreach

Events Klaviyo Créés

  • welcome_series_started → Trigger flow Welcome
  • outreach_decision_maker_started → Flow Decision Makers
  • outreach_marketing_started → Flow Marketing
  • outreach_sales_started → Flow Sales
  • outreach_tech_started → Flow Tech
  • outreach_hr_started → Flow HR

Troubleshooting

Erreurs courantes et solutions

Erreurs Klaviyo
Code Message Solution
401 Unauthorized Vérifier KLAVIYO_API_KEY dans .env
409 Duplicate profile Normal - le script update le profil existant
429 Rate limited Augmenter RATE_LIMIT_DELAY (défaut: 200ms)
400 Invalid email Filtrer les emails invalides avant push
Erreurs n8n
Erreur Cause Solution
$env not defined n8n Community ne supporte pas $env Utiliser scripts natifs .cjs avec process.env
License error: feat:variables Variables requiert licence payante Idem - scripts natifs
Workflow inactive Workflow désactivé Activer via UI n8n ou API
Erreurs Apify
Erreur Cause Solution
Insufficient credits Crédits Apify épuisés Recharger crédits
No emails extracted Actor ne trouve pas d'emails Utiliser actor premium avec email enrichment
Actor timeout Scraping trop long Réduire --max ou utiliser pagination
Erreurs Infrastructure
Erreur Cause Solution
Container restart loop Git clone échoue (repo privé) Ajouter GITHUB_TOKEN dans git clone URL
Exit code 128 Authentication GitHub manquante Format: https://{TOKEN}@github.com/...
502 Bad Gateway Container non démarré Vérifier logs: docker logs {container}
SSL certificate error Traefik certresolver Attendre propagation DNS (5-10min)
